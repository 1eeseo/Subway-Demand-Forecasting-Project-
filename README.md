### 분석과정
**1. 데이터 수집**
- 인원 적합성 
  - 서울시 지하철 시간대 별 승객 수
  - 서울시 지하철 30분 단위 이용 통계
  - 서울시 지하철 호선별, 역별 승하차 인원 정보
- 교통 접근성
  - 서울시 버스정류장 
- 위치 적합성
  - 서울시 우수 중소기업
  - 서울시 주요시설, 집객시설
  - 서울시 중,고,대학교 인원 수
 
**2. 데이터 정제**
- 컬럼 추출
- 컬럼 병합
- 변수 추출
- 변수 선정

  
**3. 분석 모델 선정**
- 현황 분석 : EDA
- 군집 분석 : K-means Clustering
- 성능평가 : LGBM

  
**4. 분석 결과**
- 그룹과 결과 적용
- 수요 예측
